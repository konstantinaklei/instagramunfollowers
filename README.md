# instagramunfollowers
An instagram bot built with module instagrapi that shows you a list with the users that dont follow you back.

                              Features
Login σε Instagram account
Fetch followers & following lists
Detect users that don’t follow back
Display usernames in terminal


                              Requirements

Python 3.9+
instagrapi
numpy
matplotlib

                             Installation
pip install instagrapi numpy matplotlib

                             Configuration
user_name = 'username'
password = '********'



Το script:
Κάνει login
Παίρνει followers & following
Υπολογίζει ποιοι δεν σε ακολουθούν πίσω
Εκτυπώνει usernames
